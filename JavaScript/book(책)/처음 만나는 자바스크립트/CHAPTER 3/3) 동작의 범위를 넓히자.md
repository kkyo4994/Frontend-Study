# 자바스크립트 문법과 기본 기능

## 동작의 범위를 넓히자

### 'no'인지 판단하기

#### else if

: if문의 조건식이 false가 되면 else 이후가 실행된다. 

[ 형식 ] => 

if(answer === 'yes'){

​	처리...

} else if(answer === 'no'){

​	처리...

} else {

​	처리...

}

> if문을 사용해서 변수에 저장돼 있는 값이나 객체의 프로퍼티 값 등을 조사하는 것을 '평가한다'라고 한다, '조사한다'이나 '확인한다' 등으로 해석하면 된다.
