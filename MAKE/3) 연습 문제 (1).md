# 연습 문제 (1)

### '수목원 체험'을 안내하는 웹 페이지

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>연습문제</title>
</head>
<body>
    <h1>체험 안내</h1>
    <ul>
        <li>운영 기간 : 5월 중순 ~ 10월 말</li>
        <li>운영 시간 : 평일 10:00 ~ 17:00/공휴일 및 주말 10:00~18:00</li>
        <li>체험 방법 : 운영 시간 내 예약 없이 체험 가능</li>
    </ul>
</body>
</html>
```



### '단체 예약 유의 사항'을 나타내는 웹 페이지

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>연습문제</title>
</head>
<body>
    <h1>단체 예약 유의 사항</h1>
    <ol>
        <li>단체 예약 인원은 <b>10인</b>부터 가능합니다.</li>
        <li><i>이용 시간은 협의 가능</i>합니다.</li>
        <li><b>인솔 교사</b>가 인솔해야 합니다.</li>
    </ol>
</body>
</html>
```



### 브라우저 실행 결과가 같은 소스를 만드시오

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>연습문제</title>
</head>
<body>
    <h1>체험 종류</h1>
    <dl>
        <dt><b>천연 허브</b></dt>
        <dd>건조한 피부를 촉촉하게 만들어주는 보습제로서 천연 식물 재료를 사용하여 피부에 전혀 자극적이지 않습니다.</dd><br>

        <dt><b>천연 비누</b></dt>
        <dd>고급 입욕제가 첨가된 조각 비누와 에센셜 오일, 천사/나비/비행기/곰 등 다양한 비누를 만드는 체험입니다.</dd><br>

        <dt><b>나무피리 목걸이</b></dt>
        <dd>천연 고급 목재인 단풍나무로 만드는 나무피리 목걸이는 막대 모양으로 간단하게 만들 수 있고 더 복잡한 모양의 목걸이도 제작 가능합니다.</dd>
    </dl>
</body>
</html>
```
