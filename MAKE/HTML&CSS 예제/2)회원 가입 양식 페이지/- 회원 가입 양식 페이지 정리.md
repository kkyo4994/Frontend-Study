# 회원 가입 양식

* 다양한 폼 양식 이용

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>회원 가입</title>
</head>
<body>
    <h3>입력 양식</h3>
    <form>
        <table>
            <tr>
                <td>이름 : </td>
                <td><input type="text"></td>
            </tr>
            <tr>
                <td>성별 : </td>
                <td><input type="radio">여성<input type="radio">남성</td>
            </tr>
            <tr>
                <td>취미 : </td>
                <td><input type="checkbox" name="hobby1">운동
                    <input type="checkbox" name="hobby1">음악
                    <input type="checkbox" name="hobby1">개발</td>
            </tr>
            <tr>
                <td><input type="submit"></td>
            </tr>
        </table>
    </form>
</body>
</html>
```
